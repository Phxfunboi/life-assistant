# Ollama Cheat Sheet

## Adding Models to Ollama

1. Open the Ollama interface.
2. Click on 'Add Model'.
3. Enter the model name and configuration details.
4. Click 'Save' to add the model.

## Managing Multiple Models

- To view all models: Go to the 'Models' tab.
- To switch between models: Select the desired model from the list.
- To remove a model: Click the 'Delete' icon next to the model name.

## Connecting Friend Dev Kit 2

1. Connect Dev Kit 2 to your computer via USB or network.
2. Open Ollama and go to 'Settings' > 'Integrations'.
3. Click 'Add Device' and select 'Friend Dev Kit 2'.
4. Follow the on-screen instructions to complete the connection.
